# LS(Living Server)

此服务为Living APP的后台服务LS, 基于golang web框架beego搭建

## Init

```bash
git clone https://github.com/Wiesen/mini_proj
```

## Run

```bash
cd mini_proj
make
bin/livingserver &
```


## Configure
```
livingserver config file: `./conf/config.toml`


beego config file: `./conf/app.conf`
```
